# xtreme-frontend

> 使用フレームワーク : Nuxt.js
> xtremeのフロントエンド用アプリケーション

## 環境構築

``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn run dev   # rails sと同じ

# build for production and launch server
$ yarn run build
$ yarn start

# generate static project
$ yarn run generate
```
